# Akarma Website

A beautiful, static multi-page website for Akarma - an organization dedicated to nurturing mind, body, and spirit through knowledge and inner peace.

## Project Structure

\`\`\`
akarma-website/
├── index.html                 # Home page
├── css/
│   └── styles.css            # All styling (vanilla CSS)
├── js/
│   └── main.js               # JavaScript functionality
├── pages/
│   ├── about.html            # About page
│   ├── classes.html          # Classes page
│   ├── publications.html     # Publications page
│   ├── counseling.html       # Counseling services page
│   ├── events.html           # Events page
│   └── contact.html          # Contact page
└── README.md                 # This file
\`\`\`

## Features

- **Fully Static**: No backend required, works on any web server
- **Vanilla HTML/CSS/JavaScript**: No frameworks or build tools needed
- **Responsive Design**: Works beautifully on mobile, tablet, and desktop
- **Clean Navigation**: Easy-to-use menu with mobile hamburger
- **Contact Form**: Functional contact form with validation
- **Smooth Scrolling**: Enhanced user experience with smooth transitions
- **Organized Structure**: Well-organized folders for easy maintenance

## Pages

1. **Home (index.html)** - Hero section with services overview
2. **About** - Mission, vision, and organization information
3. **Classes** - Meditation, yoga, and wellness class listings
4. **Publications** - Books, periodicals, and educational resources
5. **Counseling** - Holistic counseling services overview
6. **Events** - Seminars, retreats, and upcoming events
7. **Contact** - Contact form and organization information

## How to Use

1. Download or clone this repository
2. Open `index.html` in your web browser
3. Navigate through pages using the top navigation menu

## Customization

- **Colors**: Edit color variables in `css/styles.css` (`:root` section)
- **Content**: Edit text in individual HTML files
- **Images**: Add image files to an `images/` folder
- **Contact Info**: Update contact details in `pages/contact.html`

## Deployment

### GitHub Pages
1. Create a GitHub repository
2. Push this project to the repository
3. Enable GitHub Pages in repository settings
4. Website will be available at `https://username.github.io/repository-name`

### Other Hosting
- Upload the entire project folder to any web hosting service
- No special server configuration needed

## Mobile Menu

The navigation menu includes a responsive hamburger menu that appears on screens smaller than 768px. JavaScript automatically handles the toggle functionality.

## Form Submission

The contact form includes client-side validation. In a production environment, you would need to set up a backend service (like a contact form service) to actually send emails.
